# MycoTrace Public Site

This repository hosts the static public release for **MycoTrace**.

MycoTrace is a bioinformatics portfolio project focused on clinical variant triage, review-friction signals in ClinVar, and a hematology-oriented synthetic demo layer.

## Live site

- Public landing page: https://totejolo.github.io/mycotrace-site/
- Demo overview: https://totejolo.github.io/mycotrace-site/demo/
- Case explorer: https://totejolo.github.io/mycotrace-site/demo/cases/
- Gene explorer: https://totejolo.github.io/mycotrace-site/demo/genes/

## What is inside

- a public-facing landing page for presentation and outreach
- a synthetic demo bundle with case and gene exploration
- static assets ready for GitHub Pages or another static host

## Important note

This public site uses synthetic demo case data and is intended for portfolio, workflow discussion, and technical demonstration.
It is not a diagnostic system and does not replace clinical review.
